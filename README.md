## 📌 Overview

This repository contains a collection of **15 AI and Machine Learning projects** built to solve real-world problems across different domains including:

- Computer Vision  
- Natural Language Processing (NLP)  
- Deep Learning  
- Recommendation Systems  
- Time Series Forecasting

# 🚀 Projects

## 🔰 Core AI & ML Projects

| Project | Description | Tech |
|--------|------------|------|
| Face Mask Detection | Real-time mask detection using webcam | OpenCV, CNN |
| AI Career Counsellor | NLP chatbot for career suggestions | NLTK, Rasa |
| Stock Price Prediction | LSTM model for stock forecasting | Keras |
| Resume Ranker | NLP-based resume ranking system | TF-IDF |
| Plant Disease Detection | Image classification for plants | CNN |
| Fake News Detection | Classify news as fake or real | NLP |
| Sign Language Recognition | Hand gesture to text system | MediaPipe |
| Movie Recommendation | Personalized movie suggestions | ML |
| Mental Health Chatbot | Emotional support chatbot | Transformers |
| Fraud Detection | Detect fraudulent transactions | XGBoost |
| Music Genre Classification | Classify music using audio features | Librosa |
| Fitness Coach | Pose detection & rep counter | OpenCV |
| Emotion Detection (Voice) | Detect emotions from speech | ML |
| Text Summarization | Generate summaries using AI | Transformers |
| Story Generator | Generate stories using GPT | Generative AI |
